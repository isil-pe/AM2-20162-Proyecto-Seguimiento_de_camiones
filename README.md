<h3>Descripción del proyecto</h3>
</br>
<p>Seguimiento de Camiones es una app que permite registrar en tiempo real la ubicación de un camión en una base de datos mediante el uso de servicios en la nube.</p>

<h3>Funcionalidades</h3>
<ul>
<li>Autenticación</li>
<li>Mostrar listado de Ordenes</li>
<li>Registro de ubicación actualen la bd cada cierto tiempo</li>
<li>Mostrar puntos de origen y destino de la orden utilizando el API de Google Maps</li>
<li>Mostrar ruta sugerida en la aplicación Google Map</li>
</ul>
